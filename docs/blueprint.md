# **App Name**: PassKey Pro

## Core Features:

- Secure Password Generation: Secure password generation using crypto.getRandomValues().
- Breach Detection: Breach detection tool: Automatically check generated passwords against the 'Have I Been Pwned' API to flag compromised passwords.
- Encrypted Password Vault: Password storage in a local, encrypted vault.
- Password Auto-Fill: Password auto-fill feature for web forms via content script.
- Google OAuth Authentication: User authentication with Google OAuth for secure access.
- Extension Popup UI: Chrome extension popup for easy password access and generation.

## Style Guidelines:

- Primary color: Strong blue (#3F51B5), suggestive of trust and security.
- Background color: Light blue-gray (#ECEFF1), very desaturated and dark enough for comfortable reading.
- Accent color: Deep purple (#7E57C2) to add sophistication to the visuals.
- Font: 'Inter' (sans-serif) for both body and headline, creating a modern and neutral appearance.
- Use a set of minimalist line icons for password-related actions.
- Simple, card-based layout for password entries within the popup.
